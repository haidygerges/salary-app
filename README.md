# Salary App 💰

A modern, beautiful salary calculation web application built with HTML, CSS, and vanilla JavaScript. Features a glassmorphism design, login system, and multi-currency support.

## ✨ Features

- User authentication (Admin + Employee profiles)
- Salary calculation with extra hours, bonus, and penalties
- Support for multiple currencies (USD, SAR, GBP, EUR) converted to EGP
- Different multipliers for job titles (Operator, Salesman, Admin)
- Responsive glassmorphism UI
- Data persistence using LocalStorage

## 🛠 Technologies
- HTML5
- CSS3 (Glassmorphism)
- Vanilla JavaScript
- LocalStorage
- Exchange Rate API

1. Clone the repo
2. Open `login.html` in your browser
